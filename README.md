# Smart CCDGUT PHP Client
![](https://zhensheng.im/wp-content/uploads/2018/11/SmartCCDGUTPHPClient.png)
## Requirements
PHP >= 7.0

curl

json

## Usage
Login first
```
./SmartCCDGUT login [USERNAME] [PASSWORD]
```

Logout
```
./SmartCCDGUT logout
```

Query electric usage details
```
./SmartCCDGUT USERNAME [area] [room]
```
